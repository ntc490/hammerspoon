### Key-bindings

I use the hyper-key for most hammerspoon actions: shift-cmd-ctrl-alt.

| Key binding                   | Function        |
| ----------------------------- | ------------    |
| hyper-R                       | Reload          |
| hyper-w                       | Web browser     |
| hyper-t                       | Teams messenger |
| hyper-i                       | iTerm           |
